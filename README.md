
# About Project
A showcase project, to replace repo [aeolus](https://github.com/Astroleander/weboratory)

Divide into 4 part:
- algorithm
- scenario (react)
- graphics (vue)
- navigator

# Run
+ dependics:
```
node 8.6+
```
+ script:
```
npm install
npm run dev
```
and project runs on [localhost:80](http://localhost/)
